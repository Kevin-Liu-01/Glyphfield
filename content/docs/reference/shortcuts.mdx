## Global Studio search

| Shortcut | Action |
| --- | --- |
| `⌘K` | Focus Studio search on macOS |
| `Ctrl+K` | Focus Studio search on Windows/Linux |
| `/` | Focus Studio search when not editing a field |
| `Escape` | Close open project and appearance popovers |

Typing in a text input, text area, or select prevents `/` from stealing focus.

## Documentation search

Fumadocs search uses `⌘K` or `Ctrl+K` while browsing `/docs`. Search indexes headings and body content across all documentation pages.

## Canvas

The canvas toolbar exposes zoom out, a zoom slider, reset to 100%, zoom in, and fit. These controls use accessible names and can be reached with normal Tab navigation.

## Tool labels

The single-character hints in the tool catalog are compact visual mnemonics. Global Studio search is the reliable keyboard route for selecting tools.
